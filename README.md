## Stacks

- Framework: ReactJS
- UI Library: Material UI
- State management: Redux

## How to Install

0. Clone this repo
1. `cd <cloned_dir>`
2. `npm i`
3. `cp env.example .env`
4. fill your .env settings with these.

```
PORT=8080
REACT_APP_API_BASE_URL="http://localhost:3000/api/v1"
```

## How to run

1. `npm start`
